# 🎓 Student Exam Score Predictor

A modern GUI-based Machine Learning application built using Python and Tkinter to predict student exam scores based on academic inputs.

## 🚀 Features
- Predict student exam scores using Linear Regression
- Load dataset via CSV, synthetic data, or manual entry
- Model performance metrics: R², MAE, RMSE
- Visualization: coefficient chart & scatter plot
- Real-time prediction with confidence range

## 🧠 Technologies Used
Python, Tkinter, Scikit-learn, Pandas, NumPy

## 📦 Installation
pip install scikit-learn pandas numpy

## ▶️ Run
python StudentScorePredictor.py

## 📌 Inputs
Hours Studied, Attendance %, Previous Score, Assignments Done

## 📊 Output
Predicted Score, Grade, Confidence Range
